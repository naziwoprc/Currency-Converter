## ‚ú® Features

- **Real-Time Data**: Fetches latest exchange rates from the ExchangeRate API.
- **Dynamic Flag Mapping**: Automatically slices currency codes (e.g., USD -> US) to display the correct country flag.
- **Micro-Animations**: 
  - **Earth Loader**: A custom-designed SVG "rotating earth" loader for a premium wait experience.
  - **Flying Flags**: Smooth layout animations using Framer Motion when swapping currencies.
  - **Spring Physics**: Natural, bouncy transitions for UI elements.
- **Responsive Design**: Fully mobile-friendly layout built with Tailwind CSS.
